\section{Conclusion}
This project demonstrates that a Gaussian HMM is a reasonable unsupervised baseline for smartphone-based HAR and that both exact inference and Bayesian Gibbs sampling can be implemented cleanly for this setting. The toy-data sanity check validates the inference pipeline, and the HAR comparison shows that the frequentist and Bayesian versions of the same model perform similarly on held-out activity discovery. The main contribution of the Bayesian version is therefore interpretability and posterior uncertainty, rather than a decisive gain in clustering performance.
